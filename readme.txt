== Description ==

A plugin that allow you to insert advertising via shortcode.

**Plugin Features**

* Free for everyone.
* Insert advertising at anywhere you want.
* Mobile device support.
* Advertising code provided by author.

== Installation ==

1. Upload WP Shortcode Advertising to your blog.
1. Activate it.
1. Go profile options page setup your advertsing code.
1. Insert shortcode to your post/page.
